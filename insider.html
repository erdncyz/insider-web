Proje Hakkında

Bu proje, Insider şirketinin kariyer sayfasındaki QA (Quality Assurance) pozisyonlarını test etmek için geliştirilmiş Selenium WebDriver otomasyon çözümüdür.

Özellikler

  • Page Object Model (POM) tasarım deseni
  • Modern Selenium WebDriver 4.15.0
  • Otomatik ChromeDriver yönetimi
  • Optimize edilmiş element bulucular
  • Temiz ve okunabilir kod

Test Gereksinimleri

  1. Ana Sayfa: https://useinsider.com/ adresini ziyaret et ve Insider ana sayfasının açıldığını kontrol et
  2. Kariyer Sayfası: "Company" menüsünden "Careers" seç ve kariyer sayfasının açıldığını doğrula
  3. QA Departmanı: QA departmanı sayfasına git ve "See all QA jobs" butonuna tıkla
  4. Filtreleme: İşleri konum (Istanbul, Turkey) ve departman (Quality Assurance) ile filtrele
  5. Doğrulama: Tüm işlerin QA pozisyonu olduğunu ve "View Role" butonunun Lever sayfasına yönlendirdiğini kontrol et

Kullanılan Teknolojiler

  • Java 11 - Ana programlama dili
  • Selenium WebDriver 4.15.0 - Web otomasyon framework'ü
  • JUnit 4 - Test framework'ü
  • Maven - Bağımlılık yönetimi
  • WebDriverManager - Otomatik driver yönetimi

Kurulum ve Çalıştırma

Gereksinimler

  • Java 11 veya üzeri
  • Maven 3.6 veya üzeri
  • Chrome tarayıcısı

Kurulum Adımları

git clone https://github.com/erdncyz/insider-selenium.git
cd insider-selenium
mvn clean install

Test Çalıştırma

mvn test

Sonuç

Bu proje, Insider web sitesindeki QA pozisyonlarını test etmek için modern test otomasyonu en iyi uygulamalarını kullanarak geliştirilmiş kapsamlı bir çözümdür. Page Object Model tasarım deseni kullanılarak sürdürülebilir ve ölçeklenebilir test kodu sağlanmıştır.
