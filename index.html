About the Project

This project is a Selenium WebDriver automation solution developed to test QA (Quality Assurance) positions on the Insider company's career website.

Features

  • Page Object Model (POM) design pattern
  • Modern Selenium WebDriver 4.15.0
  • Automatic ChromeDriver management
  • Optimized element locators
  • Clean and readable code

Test Requirements

  1. Homepage: Visit https://useinsider.com/ and verify that the Insider homepage is opened
  2. Career Page: Select "Company" menu ‚Üí "Careers" and verify the career page is opened
  3. QA Department: Navigate to QA department page and click "See all QA jobs" button
  4. Filtering: Filter jobs by location (Istanbul, Turkey) and department (Quality Assurance)
  5. Verification: Verify all jobs are QA positions and "View Role" button redirects to Lever page

Technologies Used

  • Java 11 - Main programming language
  • Selenium WebDriver 4.15.0 - Web automation framework
  • JUnit 4 - Test framework
  • Maven - Dependency management
  • WebDriverManager - Automatic driver management

Installation and Usage

Prerequisites

  • Java 11 or higher
  • Maven 3.6 or higher
  • Chrome browser

Installation Steps

git clone https://github.com/erdncyz/insider-selenium.git
cd insider-selenium
mvn clean install

Run Tests

mvn test
